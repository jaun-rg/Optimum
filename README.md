optimum
=======

??Proyecto de Administración de Empresas de Software, en el cual generamos una aplicación web para administrar una escuela de regularización.

En  la primera fase del proyecto cuenta con módulos de Acceso y autentificación así como los módulos de administración de Alumnos y Profesores 

Para la segunda fase se incluirán los módulos de administración de Cursos, Asistencias y Calificaciones y de Administración financiera
